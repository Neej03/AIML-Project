# Smart Route Finder

## Description
Route finding is a fundamental problem in Artificial Intelligence and graph theory. It is widely used in navigation systems, logistics, and transportation networks. 
This project models cities as nodes and roads as weighted edges in a graph. Different search algorithms are applied to find paths between a source and a destination.

## Features
- BFS for minimum stops
- DFS for exploration
- UCS for minimum cost
